# helpers

Development environment setup scripts for Windows and WSL/Linux.

## Quick Start

### Windows (PowerShell — run as Administrator)

```powershell
irm https://raw.githubusercontent.com/vriveras/helpers/main/scripts/setup-windows.ps1 | iex
```

### WSL / Linux (bash)

```bash
curl -fsSL https://raw.githubusercontent.com/vriveras/helpers/main/scripts/setup-wsl.sh | bash
```
